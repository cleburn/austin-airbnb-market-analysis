# Austin Airbnb Market Analysis

Exploratory data analysis of Airbnb listings in Austin, Texas. Examines pricing patterns, neighborhood dynamics, and the impact of outliers on market statistics.

## Key Findings

- **Median nightly rate:** $132 (mean inflated to $283 by luxury listings up to $50K/night)
- **Price vs Rating:** No strong correlation; high ratings appear across all price ranges
- **Availability vs Price:** Mild negative trend; premium listings book fewer days
- **Market cluster:** $150-$200/night represents the typical Austin listing

## Tech Stack

Python · Pandas · NumPy · Matplotlib · Seaborn · Jupyter
